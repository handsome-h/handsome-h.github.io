程序员常用Mac常用快捷键

发表于2022-01-28|更新于2022-01-29|Mac效率
|阅读量:

Mac键盘上有四个通用修饰键，常用快捷键将通过这四个修饰键搭配组合

  • Command
  • Control
  • Option
  • Shift

通过系统设置里的键盘可以查看或修改指定功能的快捷键

通用

Command在大多数情况下相当于Windows下的Ctrl

  • Command + Z 撤销
  • Command + Shift + Z 重做
  • Command + X 剪切
  • Command + C 拷贝
  • Command + V 粘贴
  • Command + A 全选
  • Command + S 保存
  • Command + F 查找
  • Command + M 最小化窗口
  • Command + N 新建
  • Command + O 打开
  • Command + Shift + S 另存为
  • Command + W 关闭
  • Command + Q 退出

系统

  • Command + H 隐藏当前应用窗口
  • Command + Option + H 隐藏其他应用程序窗口
  • Command + Q 退出应用
  • Command + Option + esc 强制退出程序
  • Command + Shift + Y 将当前选中的文字生成便签
  • Command + Tab 切换程序
  • Command + Shift + Tab 反向切换程序
  • Command + ~ 在当前应用的不同窗口切换
  • Command + , 弹出当前应用配置
  • Command + Space 打开聚焦搜索
  • Command + Control + Q 锁屏

浏览器

  • Command + T 新建标签页
  • Command + Shift + T 恢复刚关闭的标签页
  • Command + N 新建窗口
  • Command + R 刷新
  • Command + F 搜索
  • Command + W 关闭当前标签页
  • Command + L 光标直接移到地址栏
  • Command + D 收藏标签页
  • Control + Tab 向右切换标签页
  • Control + Shift + Tab 向右切换标签页
  • Command + Option + ←/→ 向左或向右切换标签页
  • Command + '+' 放大页面
  • Command + '-' 缩小页面
  • Command + Option + I 打开开发者工具Network
  • Command + Option + J 打开开发者工具Console

文件管理

  • Command + Shift + N 新建文件夹
  • Command + Shift + G 调出窗口，输入绝对路径直达
  • Command + O 打开所选项，相当于鼠标双击
  • Command + ↑ 返回上一级
  • Command + Delete 将文件移到废纸篓
  • Command + Shift + Delete 倾倒废纸篓
  • Space 预览选中的文件

文本处理

  • fn + ↑ 向上滚动一页
  • fn + ↓ 向下滚动一页
  • fn + ← 跳到文档开头
  • fn + → 跳到文档结尾
  • Command + → 将光标移至当前行的行尾
  • Command + ← 将光标移至当前行的行首
  • Command + ↓ 将光标移至文稿末尾
  • Command + ↑ 将光标移至文稿开头
  • Option + → 将光标移至下一个单词的末尾
  • Option + ← 将光标移至上一个单词的开头
  • Control + A 移至行或段落的开头
