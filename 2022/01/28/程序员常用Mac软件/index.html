程序员常用Mac软件

发表于2022-01-28|更新于2022-01-29|Mac效率
|阅读量:

1、JetBrains系列

这个就不多说了，很好用的IDE，大家耳熟能详。

官网下载地址：https://www.jetbrains.com.cn/en-us/products/

2、Google Chrome

Mac自带Safari，但程序员更多的是使用Chrome，其方便调试，也有很多好用的插件，后面会再介绍。

官网下载地址：https://www.google.cn/chrome/

3、终端

Mac自带终端不是很好用，这里推荐两个，一个是iTerm2，一个Termius（跨设备）。

官网下载地址：

  • iTerm2：https://iterm2.com/
  • Termius：https://www.termius.com/

4、数据库可视化

  • Navicat 和 Sequel Pro 都是关系型数据库可视化工具；后者免费；
  • Another Redis Desktop Manager 是 Redis 数据库可视化工具；
  • Robo 3T 是 MongoDB 数据库可视化工具；

官网下载地址：

  • Navicat：https://www.navicat.com.cn/
  • Sequel Pro：http://www.sequelpro.com/
  • Another Redis Desktop Manager：https://gitee.com/qishibo/AnotherRedisDesktopManager/releases
  • Robo 3T： https://robomongo.org/

5、文本编辑器

Sublime是一个轻量的文本编辑器，Typora是一个轻量的Markdown编辑器。

官网下载地址：

  • Sublime Text：https://www.sublimetext.com/
  • Typora：https://www.typora.net/

6、接口调试

Postman是接口调试的好工具，很好用。

官网下载地址：https://www.postman.com/

7、网络抓包

Charles，俗称“花瓶”，网络抓包的好工具，支持PC和手机抓包。

官网下载地址：https://www.charlesproxy.com/

8、Word、Execel、PPT、PDF编辑器和阅读器

Mac自带Numbers表格、Keynotes讲演、Pages文稿，可能用不惯，可以用WPS来替代。PDF Expert 用来阅读PDF文件体验很不错。

官网下载地址：

  • WPS Office：https://www.wps.cn/product
  • PDF Expert：https://www.pdfexpert.cn/

9、截图工具

Mac自带截图工具，我一般使用钉钉带的截图工具，随截随粘，不过这里推荐Snappy，它可以悬浮显示截图，方便单屏场景下跨软件操作。

官网下载地址：http://snappy-app.com/

10、英语字典

可设置快捷键，想查单词的时候直接唤出即可，方便快捷。

官网下载地址：http://www.eudic.net/v4/en/app/eudic

11、文本比对

有时候需要对比文本改动地方，Beyond Compare就很方便。

官网下载地址：https://www.beyondcompare.cc/

12、远程控制

远程控制主机或笔记本，都支持移动端。两款软件都用过，这里更推荐使用向日葵。

官网下载地址：

  • TeamViewer：https://www.teamviewer.cn/cn/
  • 向日葵远程控制：https://sunlogin.oray.com/

13、系统清理

Mac轻量级系统清理软件，很好用。

官网下载地址：https://lemon.qq.com/
