Markdown格式和编辑器Typora

发表于2022-01-28|更新于2022-01-28|blog
|阅读量:

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

Markdown语法

1、标题

2、字体

3、分隔线

4、脚注

5、列表

无序列表

无序列表使用星号(****)、加号(+)或是减号(*-**)作为列表标记，这些标记后面要添加一个空格，然后再填写内容

有序列表

使用数字并加上 . 号来表示

列表嵌套

6、区块

在段落开头使用 > 符号 ，然后后面紧跟一个空格符号：

7、代码

如果是段落上的一个函数或片段的代码可以用反引号把它包起来（**`**）

代码区块

  • 使用 4 个空格或者一个制表符（Tab 键）
  • 可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）

8、链接

或者

9、图片

10、表格

使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

对齐方式

  • -: 设置内容和标题栏居右对齐。
  • :- 设置内容和标题栏居左对齐。
  • :-: 设置内容和标题栏居中对齐。

以上基本就是Markdown的全部语法，对于新手比较难记忆，所以这里推荐一个Markdown编辑器Typorea。

Typorea

Typora 是一款支持实时预览的 Markdown 文本编辑器。它有 OS X、Windows、Linux 三个平台的版本。

官网地址：https://typora.io/

通过编辑器内的快捷键轻松编写Markdown文件，且所见即所得，也可通过显示>源代码模式来显示源码。

这款软件会让Markdown写起来很轻松，大家可以下载试试。
